# 프로젝트 메모

## 목표/제약 (절대 고정)
- **목표:** UDP 141B 바이너리를 **Node UDP Bridge**에서 수신 → **WebSocket**으로 중계 → **Next.js 웹앱**에서 **파싱/표시/3D 시각화**
- **제약:** 브라우저는 UDP를 직접 수신할 수 없음 → **UDP Bridge는 필수**
- **데이터:** UDP payload = **141 bytes (바이너리)**

## 시스템 구성(요약)
- 송신 장치/프로세스 ──(UDP, 141B)──> Node UDP Bridge ──(WebSocket)──> Next.js(Web) ──> Parser ──> UI/3D
